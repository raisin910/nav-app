# ナビゲーションアプリ

## 概要
このアプリは、Next.jsとGoogle Maps APIを使用したルート案内アプリケーションです。
出発地と目的地を指定して、歩行ルートの所要時間と距離を計算します。

## 機能
- 現在地の取得と表示
- 出発地と目的地の入力（Google Places Autocomplete対応）
- 歩行ペースの設定（ゆっくり、ふつう、急いで）
- ルート表示と所要時間の計算
- 到着予定時刻の表示
